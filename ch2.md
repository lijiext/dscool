# 线性表

线性表是一个相当灵活的数据结构，对线性表的数据元素不仅可以进行存取访问，还可以 

进行插入和删除等基本操作

## ADT

| Opeartion     | Description                 |
| :------------ | --------------------------- |
| InitList      | 初始化一个空的线性表        |
| CreateList    | 根据输入的 n 个数创建线性表 |
| IsEmpty       | 判断线性表是否为空          |
| GetLength     | 获取线性表长度              |
| GetElement    | 获取位于 i 位置的元素 e     |
| LocateElement | 获取元素 e 的位置           |
| InsertElement | 在位置 i 插入元素 e         |
| DeleteElement | 删除 i 位置的元素           |
| PrintList     | 按顺序输出所有元素          |
